## ğŸš€ Getting Started

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Ruchitha-56/TaskManager_XPTracker.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd task-manager-xp-tracker
    ```
3.  **(Optional) Add a completion sound:** Place an `completion-sound.mp3` file inside an `assets` folder at the root of the project (e.g., `assets/completion-sound.mp3`).

### Usage

1.  **Open `index.html`:** Simply open the `index.html` file in your web browser.
2.  **Enter Mission Control:** Click the "Access Mission Log" button on the welcome screen.
3.  **Manage Tasks:**
    *   Use the input field to add new missions.
    *   Select a priority and an optional due date.
    *   Drag and drop tasks to reorder them.
    *   Click the checkbox to mark tasks complete (they will move to the archive).
    *   Use the edit (`<i class="fas fa-edit"></i>`) and delete (`<i class="fas fa-trash-alt"></i>`) buttons.
4.  **Switch Views:** Toggle between "Current Missions" and "Mission Archive" to see active and completed tasks.
